# Python Projects

A collection of Python projects demonstrating machine learning, data analysis, and visualization using libraries such as NumPy, Pandas, Matplotlib, and Pillow.

## Projects

### 🧠 Handwritten Digit Recognition

A feedforward neural network built from scratch using NumPy to recognize handwritten digits from the MNIST dataset. Includes a training script, an interactive drawing interface, and real-time digit prediction.

**Technologies:** NumPy, Pillow, Matplotlib

### 📊 Database Basics

A data analysis project that simulates product sales, handles missing values, calculates revenue, and visualizes business insights using bar and line charts. Covers standard usage of NumPy, pandas, and matplotlib in a single file.

**Technologies:** Pandas, NumPy, Matplotlib

## Skills Demonstrated

* Machine Learning Fundamentals

* Data Analysis & Visualization
* Data Preprocessing
